Array

An Array is a structure of ordered items, they could be of the same type or different, although the most common way to use is it with elements of the same type. The elements are stored in contiguous address or index. There are different ways to declare an array. In the block below are shown some of them.

const arr1 = new Array(10);
const arr2 = [];
let videoJuegos = ['Mario 3','Megaman','Chrono Trigger'];

Observe that in the first two initializations is the word const with this the nature of the array is locked, hence if want arr to be an integer or a string that cast will not be able to be done, hence, in this case the const locks the type of data structure, but the content of the data is able to change.

If you want to access to the a certain element, then you can use the index position, the index usually begins at 0, hence to access to 'Megaman' the correct way might be

let videoJuegos = ['Mario 3','Megaman','Chrono Trigger'];
let myGame = videoJuegos(1);

As it has been mentioned, the data type of the elements of an aray can be diffent, even there can be elements that can be a function, in the example below the element after false is an arrow function, and literal object, and even an array can be nested inside an array.

let arrayMutipleData = ['hola', 34, 1+1, true, false, ()=>{}, {a: 1},['spiderman','batman','Goku']];

In the case of nested arrays, if we would like to access to 'Goku', then the correct expression might be

let arrayMutipleData = ['hola', 34, 1+1, true, false, ()=>{}, {a: 1},['spiderman','batman','Goku']];
