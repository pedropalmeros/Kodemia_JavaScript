Strings

All strings can be seen as an array of characters, the array begins at index 0.

Consider the string Kodemia, this string begins at 0 and the letter attached to this index is K, letter o is attached to the index 1 and so on.

Consider the next code

myString = "Kodemia";

To access to each letter the string_name[], hence if i would like access to the letter i, it can be done by myString[5]

Length - Property - looks like an attribute

This method (function of an object) returns the length of the string

myString = 'Kodemia';
console.log(myString.length);
The output at the console might be 7.

chartAt(index)

With this method we can access to the index passed as argument.

myString = 'Kodemia';
console.log(myString.charAt(4));
The output at the console might be m

concat(other_string)

With this method we can concatenate two strings, consider the next code

string_1 = 'JavaScript ';

string_3 = ' :)';
string_1.concat(string_2,string_3);
console.log(string_1.concat(string_2,string_3));

The output at the console might be JavaScript Bootcamp :), note that the method is being invoked from the string_1, that is why it is the first word and then the other words in the order they are send as parameters
